# SIBERKAS

SIBERKAS adalah Sistem Informasi Berkas terintegrasi untuk proses layanan nasabah, monitoring SLA, arsip digital, transaksi administratif, dan akuntansi administratif.

## Role

- **Admin**: manajemen user, monitoring berkas/SLA, laporan operasional.
- **Customer Service**: nasabah, berkas, tracking, arsip digital, transaksi layanan.
- **Akuntan**: verifikasi transaksi, jurnal umum, buku besar, piutang, dan laporan akuntansi.

## Keunggulan versi ini

- PDF dan Excel langsung download.
- File arsip dan bukti pembayaran baru disimpan privat.
- Pembatasan percobaan login, role access, CSRF, validasi input, dan security headers.
- File SQL siap import ada di `SIBERKAS_DATABASE_IMPORT.sql`.

Baca `README_CARA_JALANKAN.md` untuk lokal dan `README_HOSTING_DAN_KEAMANAN.md` untuk deploy hosting.
